# Movie-Recommendation-System
A content-based movie recommendation system built using the MovieLens dataset.   The model recommends movies based on similarity in genres, keywords, cast, and crew metadata.

## Features
- TF-IDF vectorization of movie metadata
- Cosine similarity-based recommendation engine
- Handles over 45,000 movies
- Cleaned and preprocessed MovieLens dataset
- Built completely without user ratings (content-based approach)
- Deploy it using Streamlit

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK / Text preprocessing
- Streamlit
